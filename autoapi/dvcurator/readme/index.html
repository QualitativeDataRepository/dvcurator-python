dvcurator.readme.generate_readme(metadata, folder, token=None, key=None, repo=None)¶

Generate README.txt file.

This function uses the file README_template.txt in the repository root.

Parameters:

  • metadata (list) – Project metadata from get_metadata()

  • folder (String) – Path to QDR Prepared folder for project

  • token (string) – Dataverse token (required if the project is unpublished)

  • key (String or None) – Github API key, or None for public repository

  • repo (String or None) – Repository to use for the README template, defaults to dvcurator.hosts.curation_repo

Returns:

Path to newly generated README file

Return type:

string
